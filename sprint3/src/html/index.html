Robotic Studio 1 - Sprint 3

This is the documentation for RS1 Sprint 3.

Code Behaviour

This project encompasses SLO3.6

Installation

Step 1: Compilation

In your current working directory please run:

colcon build --symlink-install --packages-select sprint3

Step 2: Launch Gazebo and Navigation

To run gazebo and nav:

export TURTLEBOT3_MODEL=waffle_pi 
ros2 launch turtlebot3_gazebo turtlebot3_warehouse.launch.py
export TURTLEBOT3_MODEL=waffle_pi 
ros2 launch warehouse_navigation warehouse.launch.py

Step 3: Place Cylinder

ros2 run sprint3 place_cylinder --ros-args -p x:=-8.0 -p y:=-2.5

Step 4: Filter Cylinder

In a separate terminal, start ackerman

ros2 run sprint3 filter_cylinder
